# Online Book Store

This is a simple Online Book Store web application.

## Features
- User Login and Registration
- Browse Books by Category
- Add Books to Cart
- Checkout System

## Technologies Used
- HTML
- CSS
- JavaScript
- Python
- Database

## How to Run
1. Download the project
2. Open index.html in your browser
